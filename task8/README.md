# Localstack project
This project is about to parse file to files by month and year and then put them to s3 and DynamoDb

## Used services
- Localstack
- Airflow
- Python
- Pyspark
- Tableau(web version)

## Project consists of 3 parts
- Localstack 
- Airflow + Spark code
- Parsing input data
- Tableau screens

## Requirements 
All requirements are described in *requirements.txt*
